# Exploring and Visualizing the Iris Dataset

This project explores the classic Iris dataset using simple data analysis and visualization techniques. The goal is to understand the dataset structure, compare flower measurements, and see how different Iris species vary from each other.

## What This Project Does

- Loads the Iris dataset using `scikit-learn`
- Converts target labels into readable species names
- Checks the shape, columns, data types, and summary statistics
- Visualizes feature relationships using a scatter plot
- Shows feature distributions with histograms
- Uses box plots to compare feature spread and spot possible outliers

## Dataset

The Iris dataset contains measurements for three flower species:

- Setosa
- Versicolor
- Virginica

Each row includes sepal and petal measurements in centimeters.

## Tools & Libraries

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## How To Run

1. Open `exploring-and-visualizing-iris-dataset.ipynb` in Jupyter Notebook, JupyterLab, VS Code, or Kaggle.
2. Run the notebook cells from top to bottom.
3. No separate dataset download is needed because the notebook loads Iris directly from `scikit-learn`.

## Key Learning

This project is a beginner-friendly example of exploratory data analysis. It helps show how visualizations can make patterns in a dataset easier to understand before moving into machine learning.
